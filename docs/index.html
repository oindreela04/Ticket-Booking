Mark a ticket as used at the event gate
How gate validation works
The organizer's wallet signs a use_ticket transaction on Stellar. Once confirmed, the ticket is permanently marked as used on-chain — preventing double entry. The transaction is irreversible and publicly verifiable on Stellar Expert.
CLI Command
stellar contract invoke --id <CONTRACT_ID> --source organizer --network testnet -- use_ticket --organizer <ORGANIZER_ADDRESS> --ticket_id <TICKET_ID>
